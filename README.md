# Nahom Bekele - Personal Portfolio

A sleek, futuristic, and highly interactive personal portfolio website showcasing my work as a multi-talented creator in Web Development, Graphic Design, and Video Editing. 

Built with a focus on modern design aesthetics, smooth animations, and exceptional user experience across all devices.

## 🚀 Tech Stack
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) (Tabler, Simple Icons)

## ✨ Key Features
- **Modern UI:** Glassmorphism, glow effects, gradients, and a cohesive dark theme.
- **Interactive Elements:** Custom animated cursor, smooth scrolling, and scroll progress tracking.
- **Dynamic Content:** Auto-scrolling skill carousels and filterable project grids.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
